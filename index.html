Instaflick

An attempt at a pixel perfect Instagram clone

Instaflick is a photo sharing social networking service. This app allows users to view, comment, and like posts shared by other users in the Instaflick. Of course, users are also able to upload their own posts! Instaflick uses React.js & Redux for front end, and Python & Flask for back end APIs. The storage architechture is built on AWS S3. The database is built using PostgreSQL.

SoundShroud

A loose clone of the original SoundCloud

SoundShroud lets people discover and enjoy the greatest selection of music. Any Artist can create an account, and upload their song for anyone on SoundShroud to hear. As a user, you can browse SoundShroud and see what songs artists are posting and enjoy your favorite tunes! SoundShroud uses React.js & Redux for front end, and Python & Flask for back end APIs. The storage architechture is built on AWS S3. The database is built using PostgreSQL.
